# Server Stats Script

This script (`server-stats.sh`) analyzes basic server performance stats on a Linux system.

## Requirements
- A Linux server (e.g., Ubuntu, Arch, Kali, CentOS)
- Bash shell (`#!/usr/bin/env bash`)
- Basic tools: `mpstat`, `free`, `df`, `ps`, `bc`
